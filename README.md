# Colaboración para el desarrollo de modelos de visión computacional

Apoyo en la clasificación y anotación de imágenes para el desarrollo de modelos de visión computacional, orientados a la identificación de objetos, monitoreos remotos, conteo de especies
y análisis de ecosistemas marinos y costeros.

# Tipos de proyectos
|Tipos de proyectos  | Definición | Ejemplos |
|--------------------|------------|----------|
|Detección de objetos| Encuentra la ubicación de los objetos en una imagen| Detección de basura en playas: cajas, botellas, latas, plásticos.|
|Clasificación de etiqueta única| Dado un conjunto limitado de categorías, asigne una etiqueta a una imagen| Clasificación de presencia de sargazo|
|Clasificación de múltiples etiquetas| Dado un conjunto limitado de categorías, asigne una cantidad arbitraria de etiquetas que sean relevantes para la imagen| Identificación de componentes de ecosistemas marinos: coral, peces, algas…|
|Segmentación de instancias | A nivel de píxel, encuentre la ubicación de los objetos en una imagen. Máscaras individuales por objeto| Delimitación de mar, arena, cielo y cada trozo de sargazo|
|Segmentación semántica| A nivel de píxel, encuentra la ubicación de los objetos en una imagen y crea referencias únicas para cada objeto| Clasificación por clases:sargazo, mar, playa|
|Detección de puntos clave| Localiza la ubicación de objetos y sus puntos clave en una imagen. Se utiliza comúnmente para determinar la pose de un objeto.| Análisis de la línea de costa, morfología de organismos o estructuras naturales|

# Estos enfoques pueden integrarse con datos de teledetección, monitoreos de campo y base de datos estandarizadas.
